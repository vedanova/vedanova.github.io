Geolaunch

HomeGeolaunch

Geolaunch ist eine Software as a Service (SAAS) Lösung für App-Entwicklern. Diese können mit geringem Aufwand ortsbezogene Benachrichtigungen in ihren Apps zur Verfügung stellen. Gelaunch stellt hierfür die Infrastruktur über eine Schnittstelle (API) zur Verfügung.

Die SAAS Plattform musste nicht nur skalierbar sondern auch über eine entwicklerfreundliche API verfügen, die es erlaubt die Anzahl an Zugriffen zu limitieren.

Ein Beispiel für eine solche App wäre eine, welche die Position des Benutzer mitteilt und beim passieren eines Geschäftes, automatisch einen Gutschein auf das Smartphone schickt.

  • Skalierbare Software Architektur
