FME Cloud

You are here: HomeFME Cloud

For our client we have implemented a Platform as a Service (PAAS) solution that moves the companies' GIS software into the cloud. Instead of installing the software at the client, the client is now able to launch one or more provisioned servers in the cloud in minutes. The solution is built on top of the Amazon Cloud (AWS)

Our services included

  • Cloud architecture consultation
  • Software architecture
  • Automated billing
  • Front- and Backend implementation

Main technologies used

  • Rails
  • AWS: EC2, S3, DynamoDB, Route53, Cloud Formation, SES
  • Chef
  • Postgres

Project Overview

  • Client: Safe Software
  • Tasks: Software Architecture, Consultation and Development
