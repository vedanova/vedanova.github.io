FME Cloud

HomeFME Cloud

Für unseren Kunden entwickelten wir eine Platform as a Service (PAAS) Lösung, welche den GIS Server unseres Kunden in die Cloud bringt. Üblicherweise wurde dieser vor Ort, beim Klienten unseres Kunden, installiert und dort gewartet. Die PAAS Lösung ermöglicht es nun per Knopfdruck innerhalb von wenigen Minuten einen vorkonfigurierten Server zu starten und produktiv zu verwenden.

Die Lösung baut auf der Amazon Cloud (AWS) auf.

Unsere Leistungen

  • Cloud-Architektur Beratung und Entwicklung
  • Software Architektur
  • Automatisierte Kostenkalkulation und Rechnungslegung
  • Front- and Backend-Implementierung

Main technologies used

  • Ruby on Rails
  • Can.js
  • AWS: EC2, S3, DynamoDB, Route53, Cloud Formation, SES
  • Chef
  • Postgres
